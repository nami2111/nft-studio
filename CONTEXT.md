# GNStudio Context

## Domain Vocabulary

### Collection Design

The editable recipe for a generated art collection.

A Collection Design includes Project metadata, output dimensions, Layers, Traits, Ruler Trait
rules, and Strict Pair configuration. It is the user-facing concept behind the current `Project`
data type.

Use **Collection Design** when discussing the deepened module that owns editing intent and
invariants for Layers, Traits, Ruler Traits, Strict Pair settings, image resources, and persistence
dirtiness.
